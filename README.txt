# University Website Project

A static/personal/professional university website built with [HTML/CSS, JavaScript] to showcase academic accomplishments, projects, and publications.

---

##  Highlights

-  **Clear academic brand** — present your bio, research interests, and work front and center  
-  **Responsive design** — looks sharp on mobile, tablet, and desktop  
-  **Easy to maintain** — update content and publish via GitHub Pages  
-  **Structured content** — organized pages for CV, publications, upcoming events



---

##  Technologies Used

| Library / Framework | Purpose                                      |
|---------------------|----------------------------------------------|   |
| HTML5, CSS3         | Page structure and styling                   |
| JavaScript (optional) | Interactive components (e.g., gallery, forms) |
